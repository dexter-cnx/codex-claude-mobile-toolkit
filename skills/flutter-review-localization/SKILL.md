# flutter-review-localization

## Purpose

Review localization correctness for a Flutter/mobile change.

## Review focus

- user-facing text handling
- CSV-first source-of-truth adherence
- generated output expectations
- dotted-key consistency
- untranslated or hardcoded string risks

## Expected outputs

- localization findings
- key risks
- generation or regeneration expectations

## Verification

If localization files were changed, state whether regeneration was run, expected, deferred, or unavailable.
