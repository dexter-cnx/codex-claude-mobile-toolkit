# flutter-review-platform-config

## Purpose

Review platform and configuration safety for Flutter/mobile changes.

## Review focus

- Android, iOS, macOS, and web config changes
- native boundary impact
- scheme or target assumptions
- environment or build-script changes
- risk of platform-specific regressions

## Expected outputs

- platform findings
- validation expectations
- unverified platform risks

## Verification

When platform files were touched, include the exact files reviewed and the target validation that should be run.
