# เส้นทางเริ่มต้น

หน้านี้คือทางลัดสำหรับคนที่เพิ่งเริ่มใช้ toolkit นี้

## ถ้าคุณมีเวลาอ่านแค่ 5 นาที

อ่านตามนี้:

1. `AGENTS.md`
2. `README.md`
3. `docs/skill-workflow-map.md`
4. เลือก prompt ตัวแรกจาก [[03-คู่มือ Prompts]]
5. เลือก skill ที่เข้าคู่จาก [[04-คู่มือ Skills]]

## วิธีคิดที่ถูกต้องก่อนใช้

Toolkit นี้ไม่ได้บอกให้ "เริ่มเขียนโค้ดทันที"

ลำดับที่ถูกคือ:

1. อ่านกติกา
2. เลือก workflow หรือ prompt
3. เลือก skill ที่เหมาะ
4. ทำงาน
5. verify
6. update memory
7. handoff หรือ finalize

## เส้นทางแนะนำสำหรับผู้ใช้ใหม่

### Beginner path

1. อ่าน [[01-แผนที่ repo และส่วนประกอบ]]
2. อ่าน [[02-โฟลว์ทำงานมาตรฐาน]]
3. เปิด [[03-คู่มือ Prompts]]
4. ทำงานจริงด้วย workflow เล็กหนึ่งงาน

### Intermediate path

1. เรียนรู้ [[04-คู่มือ Skills]]
2. ใช้ [[05-สูตรใช้งานตามสถานการณ์]]
3. เริ่มใช้ `project_memory/`

### Advanced path

1. ใช้ Antigravity workflows
2. ใช้ runtime matrix
3. ใช้ stitch handoff
4. ใช้ coherence pass และ self improve toolkit

ไปต่อ: [[01-แผนที่ repo และส่วนประกอบ]]
