# Workflow: Build Optimize

## Use when
Improving build reliability, speed, or project build configuration quality.

## Steps
1. Inspect current build setup and scripts.
2. Identify bottlenecks or noisy configuration.
3. Make small, reversible build/config changes.
4. Re-run the relevant build/check commands.
5. Report measurable or observable impact.

## Required verification
- relevant build command for the target platform
- `flutter analyze`
- note any environment limitations clearly
