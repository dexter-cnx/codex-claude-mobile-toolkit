# Self Improve Toolkit

## Objective
Capture reusable lessons and route them into the correct permanent toolkit assets.

## Phases
1. identify lesson
2. test reusability
3. choose destination
4. prefer update over new file
5. mirror to Antigravity when appropriate
6. generate improvement log

## Required artifacts
- improvement log
- destination rationale

## Suggested supporting skills
- `flutter-self-improve`
