# Investigate Flutter Bug

## Objective
Run a structured bug investigation before speculative fixing.

## Phases
1. define symptom
2. define reproduction
3. isolate likely layer
4. generate hypotheses
5. inspect code evidence
6. identify root cause
7. define smallest safe fix
8. define regression checks
9. generate investigation artifact

## Required artifacts
- root-cause analysis
- proposed smallest safe fix
- regression checklist

## Suggested supporting skills
- `flutter-bug-investigation`
- `flutter-review-architecture`
- `flutter-review-testing`
