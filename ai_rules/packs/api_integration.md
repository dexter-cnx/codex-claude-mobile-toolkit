# Task Pack: API Integration

Use this pack when integrating a REST or service API.

## Rules
- Add DTOs in `data`.
- Add explicit mappers from DTOs to entities.
- Keep repository contracts in `domain`.
- Translate transport errors into app-level failures.
- Keep networking code and business logic separate.

## Deliverables
- API client changes
- DTOs
- mappers
- repository implementation
- use cases if needed
- presentation wiring
- test notes
