# Review Flutter UI prompt

Run a structured review of a Flutter screen, feature, or design implementation.

## Review dimensions
- objective clarity
- hierarchy
- component consistency
- spacing and rhythm
- action prioritization
- state coverage
- localization resilience
- accessibility
- responsiveness or small-screen fit
- maintainability of the implementation

## Output
Use `templates/ux-audit-report.md` when possible.

At minimum provide:
1. findings
2. severity
3. recommended fixes
4. implementation complexity notes
5. verification notes
