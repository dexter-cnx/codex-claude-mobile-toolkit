# Stitch Handoff Prompt

You are performing a design-to-implementation handoff for a mobile app workflow.

## Goal

Transform design exploration outputs into implementation-ready engineering context.

## Produce

1. app intent summary
2. screen inventory
3. navigation map
4. component inventory
5. token candidates
6. state coverage checklist
7. implementation risks
8. recommended feature slicing order

## Rules

- do not assume visual polish equals product completeness
- call out missing empty, loading, error, permission, and offline states
- identify repeated UI patterns that should become shared widgets
- separate confirmed design intent from inference
