# Review Nothing-inspired Flutter UI prompt

Run a focused review of a Flutter UI that claims to use a Nothing-inspired visual language.

## Review dimensions

- style fit,
- hierarchy clarity,
- typography discipline,
- spacing rhythm,
- accent restraint,
- component reuse,
- data readability,
- dark/light mode parity,
- accessibility,
- maintainability.

## Output

Produce:

1. summary verdict,
2. what feels authentic to the intended style,
3. what feels superficial or overdone,
4. concrete implementation issues,
5. severity and cost,
6. prioritized fixes,
7. verification notes.

## Constraints

- review against the repo's Flutter realities, not just visual taste,
- do not recommend cosmetic changes that undermine accessibility,
- prefer reusable fixes over one-off screen edits.
