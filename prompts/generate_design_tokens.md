# Generate design tokens prompt

Generate or refine a Flutter-friendly token proposal for the target product or feature.

## Output
Produce:
1. token categories needed
2. suggested color roles
3. typography scale
4. spacing scale
5. radius and elevation guidance
6. component usage notes
7. theme integration notes
8. migration notes if existing tokens already exist

## Constraints
- align with Material 3 unless repo context clearly overrides it
- prefer semantic roles over one-off hardcoded values
- keep token recommendations realistic for Flutter implementation
