# Prompt: Accessibility Pass

Run an accessibility-focused review on the current feature, screen, or change set.

You must inspect:
- semantics,
- text scaling resilience,
- contrast risks,
- interactive target sizing,
- empty/loading/error state clarity,
- focus order or keyboard behavior when relevant.

Use:
- `docs/repo_discipline/accessibility-standard.md`
- `templates/checklists/accessibility-review.md`

Output:
- Scope Reviewed
- Findings
- Severity
- Recommended Fixes
- Quick Wins
