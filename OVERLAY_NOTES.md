# Overlay Notes

This package is an overlay for `dexter-cnx/codex-claude-mobile-toolkit`.

## Intended usage

1. Unzip at the root of the target repository.
2. Review file replacements and additions.
3. Run markdown formatting or your preferred docs linting if used.
4. Commit as a documentation and workflow hardening update.

## What is included

- production-grade root README
- expanded root operating entrypoints
- stronger workflow docs
- stronger pipeline prompts
- new workflow-oriented skills
- reusable artifact templates

## What is not included

This overlay does not attempt to rewrite every existing skill or doc in the repository.
It focuses on the top-level navigation, routing, and production hardening layer.
