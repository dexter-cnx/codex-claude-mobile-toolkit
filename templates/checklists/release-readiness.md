# Release Readiness Checklist

## Change summary
- [ ] Scope summarized
- [ ] Affected flows listed

## Validation
- [ ] Format checked
- [ ] Static analysis checked
- [ ] Relevant tests checked
- [ ] Build impact reviewed
- [ ] Localization impact reviewed

## Product quality
- [ ] Accessibility impact reviewed
- [ ] Error handling reviewed
- [ ] Empty states reviewed
- [ ] Loading states reviewed

## Risk
- [ ] Risk level assigned
- [ ] Residual risks listed
- [ ] Rollback or mitigation noted when needed
