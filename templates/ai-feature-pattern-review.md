## Review questions

1. Could a user mistake inference for fact?
2. If the system recommends or ranks, can the user understand why?
3. If the workflow is interrupted, can it be resumed with context intact?
4. Are we adding a pattern because it solves a real problem, or because it sounds sophisticated?
5. Does the implementation remain appropriate for the platform and audience?
