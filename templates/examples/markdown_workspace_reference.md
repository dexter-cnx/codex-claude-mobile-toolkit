# Markdown Workspace Reference Pattern

This is a reference-only pattern for apps inspired by markdown knowledge-base tools.

## Reusable capability slices
- offline-first repository pattern
- local FTS search
- optional local-first sync
- export / backup architecture
- secure local data pattern

## Keep this out of core by default

Do not treat markdown workspace behavior as required core toolkit behavior.
It should remain a feature pack or project-specific overlay unless the target product is explicitly a note/wiki/PKM style app.
