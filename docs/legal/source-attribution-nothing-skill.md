# Source attribution: nothing-design-skill

## Upstream source

This overlay pack was informed by the public repository:

- `dominikmartn/nothing-design-skill`

## What was reused conceptually

The pack adapts these upstream ideas into a Flutter-oriented toolkit context:

- the style philosophy,
- typography-led hierarchy,
- monochrome-plus-signal-color token philosophy,
- component motifs such as segmented progress and mono metadata labels,
- explicit dark/light mode treatment.

## What was not copied directly

This pack does not attempt to mirror the upstream package structure or its platform mappings for HTML/CSS and SwiftUI. It reframes the ideas for Flutter, toolkit routing, prompts, and semantic tokens.
