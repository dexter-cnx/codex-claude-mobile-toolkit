# Repo Integration Playbook

## Objective
Integrate toolkit assets into an existing Flutter + Flame repo with minimal disruption.

## Order
1. Audit repo
2. Add/align AGENTS rules
3. Add telemetry sink/service
4. Hook level lifecycle events
5. Hook move/wait/undo/restart/fail/complete
6. Align JSON loading
7. Add smoke tests
8. Document paths and conventions
